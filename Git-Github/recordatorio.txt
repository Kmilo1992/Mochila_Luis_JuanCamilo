| /////////////////    COMANDOS GENERALES DE BASH    /////////////////////// |
|----------------------------------------------------------------------------|
| $ mkdir folder             | Crea una nueva carpeta                        |
|----------------------------------------------------------------------------|
| $ touch file.ext           | Crea fichero o archivo y su extension         |
|----------------------------------------------------------------------------|
| $ ls                       | Lista la carpeta                              |
|----------------------------------------------------------------------------|
| $ cd folder                | Entrar a un directorio                        |
|----------------------------------------------------------------------------|
| $ cd ..                    | Regresar al directorio anterior               |
|----------------------------------------------------------------------------|
| $ rm file                  | Elimina un archivo                            |
|----------------------------------------------------------------------------|
| $ mv file                  | Mueve de carpeta un archivo o carpeta         |
|----------------------------------------------------------------------------|
| $ rm folder                | Elimina una carpeta vacia                     |
|____________________________________________________________________________|

 _____________________________________________________________________________
| /////////////////    COMANDOS PARA UTILIZAR GIT    ///////////////////////  |
|-----------------------------------------------------------------------------|
| $ git init                      | Inicializa el repositorio y crea el .git  |
|-----------------------------------------------------------------------------|
| $ git remote add origin URL     | Vincula el repositprio local con github   |
|-----------------------------------------------------------------------------|
| $ git config --global user.name "user"     | Configura el usuario           |
|-----------------------------------------------------------------------------|
| $ git config --global user.email "email"   | Configura el email             |
|-----------------------------------------------------------------------------|
| $ git add file                  | Agrega un archivo al Staging Area         |
|-----------------------------------------------------------------------------|
| $ git add .                     | Agrega todo al Staging Area               |
|-----------------------------------------------------------------------------|
| $ git status                    | Muestra el status del repositorio         |
|-----------------------------------------------------------------------------|
| $ git commit -m "message"       | Ejecuta el commit para enviar al repo..   |
|-----------------------------------------------------------------------------|
| $ git log                       | Muestra un historial de commits           |
|-----------------------------------------------------------------------------|
| $ git pull                      | Trae al repositorio local el de github    |
|-----------------------------------------------------------------------------|
| $ git push                      | Sube a git hub el repositorio local       |
|-----------------------------------------------------------------------------|
| $ git branch "name"             | Crea una nueva rama                       |
|-----------------------------------------------------------------------------|
| $ git branch checkout "name"    | Cambia la rama de trabajo y AD            |
|-----------------------------------------------------------------------------|
| $ git branch                    | Muestra una lista de las ramas            |
|_____________________________________________________________________________|
